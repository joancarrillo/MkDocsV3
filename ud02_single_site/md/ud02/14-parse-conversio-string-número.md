# 14. Parse (conversió String → número)


---

<small class="small">UD02 · DAM · 2025</small>


| Tipus  | Classe    | Exemple                         |
|--------|-----------|----------------------------------|
| `int`  | `Integer` | `Integer.parseInt("123")`       |
| `double` | `Double`| `Double.parseDouble("12.3")`    |
| `float` | `Float`  | `Float.parseFloat("1.5")`       |
| `boolean` | `Boolean` | `Boolean.valueOf("true")`   |

```java
String s = "123";
int num = Integer.parseInt(s);
```
